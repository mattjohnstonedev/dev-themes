# Dev Themes

A curated monorepo of custom visual themes for various development applications, terminals, and code editors. Each theme is organized by application and designed for easy installation and a consistent, polished user experience.

## Supported Applications & Themes

## Contributing

Contributions are welcome! To add a new theme:

1. Create a new subdirectory under the relevant application folder (e.g., `windterm/my-new-theme/`).
2. Include all required theme files for the target application.
3. Update the application-specific `README.md` with your theme entry.
4. Optionally, add a screenshot to `assets/screenshots/` using the naming convention `[app-name]-[theme-name].png`.
5. Submit a pull request with a clear description of the theme and its source inspiration.
